# מצפן הבחירות 2026 🧭

שאלון התאמה פוליטית: שמונה שאלות ליבה, שתי רמות ניסוח (בגובה העיניים / מעמיק), ותוצאה שמראה לאילו מפלגות העמדות שלך הכי קרובות.

בנוי ב־**React 18 + Vite + Tailwind CSS**.

---

## הפעלה מהירה

צריך [Node.js](https://nodejs.org) בגרסה 18 ומעלה.

```bash
npm install     # התקנת התלויות (פעם אחת)
npm run dev     # הרצה בסביבת פיתוח → http://localhost:5173
```

### בנייה לפרודקשן

```bash
npm run build   # יוצר תיקיית dist/
npm run preview # תצוגה מקדימה של הבנייה
```

תיקיית `dist/` היא סטטית לגמרי — אפשר להעלות אותה כמו שהיא ל־Netlify, Vercel, GitHub Pages, Cloudflare Pages או כל שרת סטטי.

## איך לערוך את התוכן

הכל נמצא ב־`src/ElectionsCompass.jsx`, בראש הקובץ:

### מפלגות — `PARTIES`
לכל מפלגה:
- `name` — שם התצוגה
- `hex` — צבע (משפיע על הגרפים, הציר וכרטיס המנצחת)
- `position` — מיקום על ציר שמאל(0)–ימין(100)
- `description` — התיאור שמופיע בתוצאות

### שאלות — `QUESTIONS`
לכל שאלה:
- `category`, `icon` (מ־lucide-react), `accent` — צבע הקטגוריה
- `variants.simple` / `variants.advanced` — שני ניסוחים, כל אחד עם 4 תשובות
- `stances` — עמדת כל מפלגה בסולם **0–3**, כאשר האינדקס מתאים לסדר התשובות במערך `options`. אפשר להשתמש בשברים (למשל `1.6`) כדי לבטא עמדת ביניים.

> ⚠️ סדר התשובות ב־`options` חייב להתאים לסולם ב־`stances`, וגם בין `simple` ל־`advanced` (אפשרות 0 ב־simple = אפשרות 0 ב־advanced).

### לוגיקת הניקוד
```
דמיון = 1 - |תשובת המשתמש - עמדת המפלגה| / 3
התאמה = Σ(דמיון × משקל) / Σ(משקל) × 100
```
המשקל נקבע לפי בורר החשיבות בכל שאלה: שולי `0.5`, רגיל `1` (ברירת מחדל), קריטי `2`. שאלות שדולגו לא נכללות בחישוב.

## עיצוב ונגישות

- **טיפוגרפיה:** Secular One לכותרות, Assistant לטקסט הרץ.
- **RTL מלא**, כולל מקרא המפה הפוליטית שמיושר לציר (שמאל פוליטי = צד שמאל פיזי).
- **רספונסיבי:** במובייל מוצג לוח ההתאמות; המפה הפוליטית האופקית מוצגת מרוחב `lg` ומעלה.
- מכבד `prefers-reduced-motion` — כל האנימציות נכבות עבור משתמשים שביקשו זאת.
- טבעות פוקוס לניווט מקלדת בכל האלמנטים האינטראקטיביים.
